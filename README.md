# Spark Clean

Spark Clean is a home and office cleaning website that helps users book professional cleaning services easily and quickly.

## Features
- Home cleaning services
- Office cleaning services
- Online booking and scheduling
- Contact form for inquiries
- Responsive design

## Purpose
Spark Clean makes it easier to maintain clean, healthy, and organized spaces.
